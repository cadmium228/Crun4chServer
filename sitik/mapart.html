Введение и подготовка (Нагло украдено с svinland.ru/wiki/arty/kak-sozdat-art)

Это пошаговое руководство поможет вам создавать качественные арты на картах (Map Art) с помощью мода Litematica и онлайн-инструмента MapartCraft.

Что вам понадобится:

  • Изображение: Картинка, которую вы хотите превратить в арт
  • Сайт: MapartCraft для создания схемы
  • Мод: Litematica для отображения схемы в игре

Работа с MapArtCraft

1. Загрузка изображения

  1. Подготовьте и сохраните изображение на компьютере
  2. Перейдите на MapartCraft
  3. Нажмите на поле для загрузки и выберите ваш файл

2. Настройка схемы

  • Шаблон блоков: Выберите "Коврики"
  • Версия: 1.20 и выше
  • Размеры карты: Укажите количество карт (1x1, 2x1, 2x2)
  • Добавление блоков: Оставьте cobblestone как основу
  • Добавить блоки снизу: Выберите "Важные блоки"

3. Обработка изображения

  • Ступенчатость: 2D (простой вариант)
  • Смешивание: "Без смешивания" для четких линий
  • Предварительная обработка: Обязательно включите
  • Настройте яркость, контраст и насыщенность

4. Скачивание схемы

  • Проверьте список необходимых блоков
  • Скачайте схему (если выбрать "Скачать участок 1x1.zip" то Мапарт скачается частями по 1x1)
  • Переименуйте файл для удобства

Работа с Litematica

1. Установка схемы

  1. Переместите .nbt файл в папку ./schematics/
  2. В игре откройте меню Litematica (клавиша M)
  3. Выберите "Загрузить схему"
  4. Разместите схему в мире

2. Позиционирование

  • Включите сетку чанков (F3 + G)
  • Возьмите палку и переключите на Edit Schematic (Ctrl + Колесико)
  • Расположите схему так, чтобы левый верхний угол совпадал с углом чанка
  • Вспомогательная полоса из булыжника должна быть за границей чанка

3. Настройка мода

  • Easy Place Mode: Включите в Configuration Menu
  • Назначьте удобную клавишу для easyPlaceToggle
  • Послойное строительство: Используйте Set Here
  • Переключайтесь между слоями (PageUp/PageDown)

Строительство и финализация

1. Ручное редактирование

Исправляйте одиночные неправильные пиксели вручную во время строительства.

2. Дюп ковров

Машина для дюпа ковров (фотку в интернете найдешь):

  • 1x Наблюдатель
  • 1x Веерный коралл
  • 1x Рычаг
  • 1x Липкий поршень
  • 5x Блок слизи
  • 1x Вспомогательный блок

Важно: Выключайте машину, когда не используете. Намеренное создание лагов наказуемо.

3. Финальные шаги

  1. Полностью снесите арт и платформу после завершения
  2. Возьмите пустую карту и нажмите ПКМ в чанке
  3. Повесьте готовую карту в рамку
